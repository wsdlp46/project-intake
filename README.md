# project-intake

项目材料入库 Skill，新文件或会议记录进入项目时自动完成标准化处理。

## 功能

- 二进制文件自动转换为可读的 MD 格式
- 会议时间线自动更新
- 关键决策提炼入库
- 项目索引自动更新

## 触发

说"入库""归档""把这个文件整理一下""把这份会议记录记进去"等，或直接提供新文件路径/粘贴内容。

## 安装

```bash
cp -r project-intake ~/skills/
```
